## Quick start
### 1) API
```
cd server
# Windows
run.bat
# macOS/Linux
bash run.sh
```
Your API will be on http://localhost:8000

### 2) Client
Open a new terminal:
```
cd client
npm install
npm run dev
```
Visit http://localhost:5173

The client is already wired to the API via `VITE_API_BASE` in `client/.env`.
